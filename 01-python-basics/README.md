# תכנות אלגוריתמים מחקריים- החלק התכנותי
## מבוא לשפת פייתון-  

בחלק זה נתעמק בנושאים הבאים:

- מבוא לשפה:
   - עקרונות השפה
   - מערכת ניהול החבילות של פייתון
   - jupyter notebook
- מבנה של תוכנית פייתון:
     - סקריפט מול module 
     -  ייבוא מודולים 
     - מוסכמות

- אופרטורים:
   - אריתמטיים
   - השמה
   - השוואה
   - לוגים
   - זהות ושייכות
   - bitwise
- טיפוסי נתונים:
   - מספרים
   - boolean
   - מחרוזות
   - מבני נתונים
   - NoneType
   - mutable ו- immutable
   - המרות.
- בקרות זרימה (תנאים ולולאות)
- פונקציות:
   - פרמטרים immutable  ו- mutable
   - סקופ של משתנים 
   - *args **kwargs
   - annotations
   - פונקציות למדא
   - פונקציות built-in שימושיות.
- קבצים וחריגות:
   - טיפול בקבצים (קריאה וכתיבה)
   - המילה השמורה with
   - חריגות וטיפול בהן
